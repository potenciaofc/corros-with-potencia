Os maiores dos EUA

A Ford, Chevrolet e Dodge têm uma das maiores concorrências do mundo automotivo, entre os seus modelos os principais são:

Camaro (Chevrolet);
Mustang (Ford);
Charger R/T (Dodge);

Os modelos são concorrentes entre si, todos com motor V8 de fábrica, verdadeiras máquinas com tração traseira, com uma média de 220km/h, hoje parece pouco, mas para a época era incrível!!!
